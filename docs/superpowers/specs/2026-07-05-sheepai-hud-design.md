# SheepAI HUD — 设计文档

**日期**: 2026-07-05
**项目**: sheepai-hud — 小羊AI中转站桌面仪表盘

---

## 概述

一个 macOS 原生 SwiftUI + WidgetKit 应用，用于直观查看小羊AI中转站平台的用户信息和令牌使用情况。包含宿主 App（仪表盘+设置）和 3 种桌面小组件。

## API 接口

所有 API 基础 URL: `https://www.sheepai.top`

| API | Method | URL | Headers | 用途 |
|-----|--------|-----|---------|------|
| 获取账号信息 | GET | `/api/user/self` | `new-api-user` (用户ID), `Authorization` (系统令牌) | 用户总览 |
| 获取令牌列表 | GET | `/api/token/?p=0&size=N` | `new-api-user` (用户ID), `Authorization` (系统令牌) | 令牌列表 |
| 获取令牌使用情况 | GET | `/api/usage/token/` | `Authorization: Bearer <API令牌>` | 具体令牌监视 |

## 数据模型

### 凭证存储
- 用户 ID (`new-api-user`) 和系统令牌 (`Authorization`) 由用户手动填写
- 存储在 App Group 共享的 UserDefaults 中
- 所有小组件通过 App Group 读取同一份凭证

### 余额换算
- 比例: **500,000 系统单位 = 1 美元**（硬编码常量，可修改）
- 所有 quota / used_quota / remain_quota 展示时自动换算

## 架构

```
┌─────────────────────────────────────────────────┐
│              App Group UserDefaults              │
│  (凭证 + 缓存API响应数据)                         │
└──────┬──────────────────────────┬───────────────┘
       │                          │
┌──────▼──────────┐    ┌──────────▼──────────────┐
│   宿主 App       │    │      WidgetKit 小组件     │
│   SwiftUI        │    │                         │
│                  │    │  ┌─ 用户总览 (小号)      │
│  ┌─ 用户总览页   │    │  ├─ 令牌总览 (小号)      │
│  ├─ 令牌列表页   │    │  └─ 令牌监视 (中号)      │
│  └─ 设置页       │    │                         │
│                  │    │  Timeline 刷新 / 点击    │
│  URLSession 请求  │    │  打开宿主 App            │
└──────────────────┘    └─────────────────────────┘
```

## 宿主 App 页面

### 1. 用户总览页
- 用户名 + 邮箱
- 余额（美元）
- 已使用额度（美元）
- 请求次数
- 令牌数量

### 2. 令牌列表页
- 表格展示所有令牌：名称、状态、已用额度、分组
- 点击可查看详情
- 手动刷新按钮

### 3. 设置页
- 用户 ID 输入框
- 系统令牌输入框
- 保存按钮（写入 App Group）

### 令牌监视小组件 (systemMedium)
- 显示选中令牌的名称、无限量/有限额状态
- 已用额度、剩余额度
- 上次访问时间
- 所属分组
- 无限额时: 显示 "🟢 无限量" + 已用额度
- 有限额时: 显示 "⚠️ 有限额" + 已用额度 + 剩余额度

## 数据刷新

- **宿主 App**：前台时自动刷新 + 手动下拉刷新，结果写入 App Group 缓存
- **小组件**：Timeline 每 15 分钟自动从 App Group 读取缓存数据
- **用户点击小组件** → 打开宿主 App 触发即时刷新

## 错误处理

- 凭证未配置 → "请先打开 App 设置凭证"
- 网络错误 → 显示上次成功缓存 + 时间戳提示
- API 错误 → App 内显示错误详情

## 技术栈

- SwiftUI + WidgetKit（原生，无第三方依赖）
- 最低支持: macOS 14 (Sonoma)
- Swift 目标版本: 5.9+
- 构建工具: Xcode 16+
